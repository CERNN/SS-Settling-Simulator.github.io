==================
Casos de validação
==================

Os casos de validação são essenciais para garantir que o software produza resultados **confiáveis e consistentes**, refletindo com precisão os fenômenos físicos subjacentes.
Ao realizar esses casos de teste, podemos verificar se o software está adequadamente calibrado e é **capaz de reproduzir comportamentos esperados** em uma variedade de cenários.

Sedimentação em Fluido Power Law
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

O primeiro caso de validação refere-se a sedimentação de partículas de barita em um fluido do modelo de potência.

Os resultados dos perfis de concentração ao longo do tempo de ensaio, foram reportados por :footcite:t:`rocha2020settling`.

Para visualizar os resulados desse caso, acesse o `notebook do caso de validação <./rocha_2020.ipynb>`_.

Esse notebook contém o pós-processamento a partir dos dados da simulação.
Para consultar como foi executada a simulação, acesse a seção da documentação sobre o `módulo de simulação <../simulation/index.rst>`_.
