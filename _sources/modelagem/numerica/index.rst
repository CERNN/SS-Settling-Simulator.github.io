==================
Modelagem Numérica
==================

Esta seção aborda a metodologia numérica para a solução do modelo matemático baseado na **Teoria das Misturas da Mecânica do Contínuo**.

Uma breve descrição do modelo matemático, e do equacionamento utilizado pode ser consultado na seção `Modelagem Matemática <../matematica/index.rst>`_.

A seção de modelagem numérica é divida em:

* `Discretização Espacial <espacial.rst>`_: Discretização das equações utilizando o Método dos Volumes Finitos.
  
* `Discretização Temporal <temporal.rst>`_: Integração temporal utilizando o método de Runge-Kutta de 4ª ordem.
  
* `Parâmetros Numéricos <parametros.rst>`_: Processo estatístico para a estimação dos parâmetros numéricos e elementos de malha.
